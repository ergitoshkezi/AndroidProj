<task type="auto">
  <name>Task 4: Create ConfidenceEngine with weighted multi-factor scoring</name>
  <files>Ingredient/app/src/main/java/com/example/ingredient/parser/confidence/ConfidenceEngine.kt</files>
  <read_first>Ingredient/app/src/main/java/com/example/ingredient/parser/confidence/ConfidenceEngine.kt</read_first>
  <action>
Created class ConfidenceEngine with score(ast: MenuAST, totalLines: Int, noiseLines: Int): MenuAST method.

Implemented multi-factor confidence scoring:

Section scoring:
- sConf = 0.40 * header.confidence + 0.45 * avg(item.confidence) + 0.15 * sigmoid(itemCount/5)

Item scoring via scoreItem():
- cName: 1.0 if 3-60 chars & 1-8 words, 0.7 if 3-80 chars & 1-12 words, else 0.3
- cPrice: 1.0 for INLINE_SAME_LINE/DOT_LEADER, 0.9 for NEXT_LINE_STANDALONE, 0.7 for COLUMN_ALIGNMENT, 0 for missing
- cContext: 1.0 if section header confidence >= 0.7, scaling down to 0.3
- cLength: 1.0 for 2-8 words, 0.7 for 1 or 9-12, else 0.3
- Item conf = 0.35*cName + 0.30*cPrice + 0.25*cContext + 0.10*cLength

Overall scoring:
- cStructure: fraction of sections with confidence >= 0.5
- cPriceCov: fraction of items with price
- cNoise: 1 - (noiseLines / totalLines)
- cSize: 1.0 if 1-30 sections & 1-300 items, else 0.5
- overall = 0.35*cStructure + 0.25*cPriceCov + 0.25*cNoise + 0.15*cSize

Returns AST with populated ASTConfidence (overall, structureScore, priceScore, categoryScore, noiseScore, breakdown map).
Overall confidence used for D-15 fallback decision (< 0.5 triggers legacy).
