<objective>
Establish foundation types and utilities for the deterministic menu parser pipeline.

Purpose: Create the data model layer (MenuAST, TypedLine tokens), regex patterns (RegexRegistry), and locale support (LocalePackRegistry) that all subsequent pipeline layers depend on. Per D-04/D-05, MenuAST is internal representation with confidence scores that are discarded before mapping to List<MenuCategory>.

Output: Four foundation files in parser subpackages enabling lexer and grammar parser construction.
</objective>
