3. MenuGrammarParser — state machine consuming tokens and building AST

Output: Three files implementing Layers 1-2 of the 6-layer pipeline.

Created class WindowedContextResolver(mode: ParsingMode) with resolve(tokens: List<TypedLine>): List<TypedLine> method.

Implemented context-aware resolution:
- resolveAmbiguous: examines ±3 line window, prev/next non-blank tokens
  - Computes dishScore based on: prev is CategoryHeader (+0.40), next is PriceLine (+0.50), next is DescriptionLine (+0.30)
  - Computes headerScore based on: prev is SectionDivider (+0.50), next is DishCandidate (+0.40), window has 2+ dishes (+0.35)

- refineDish: boosts DishCandidate confidence by +0.15 if followed by PriceLine within 3 lines

- refineHeader: boosts CategoryHeader confidence by +0.20 and adds FOLLOWED_BY_DISHES indicator if 2+ dishes follow within 5 lines

Created class MenuGrammarParser(mode: ParsingMode = ParsingMode.BALANCED) with parse(tokens: List<TypedLine>, metadata: ParseMetadata): MenuAST.

Implemented 6-state machine:
- INIT: looking for first header or dish
- HEADER_FOUND: header seen, expecting dishes
- IN_SECTION: inside a section, processing items
- DISH_OPEN: dish name captured, awaiting price/description/next dish
- DESC_COLLECTING: collecting multi-line descriptions
- DONE: parsing complete

State transitions tracked via stateTransitions list for debugging/tracing.

Key behaviors:
- commitDish(): calculates dish confidence based on name length, price presence, description; respects mode.itemConfidenceFloor (D-13)
- commitSection(): finalizes section with average item confidence; respects mode.headerConfidenceFloor (D-14)
- Orphan handling: dishes before first header go to orphanedItems list
- ENRICH_THRESHOLD = 0.65f — sets enrichmentAllowed flag on high-confidence dishes (D-12)
- Blank line handling: 2+ consecutive blanks can close dish context
- Divider handling: dashes/equals commit current section and expect new header

Returns MenuAST with sections, orphanedItems, metadata, trace (events + classificationBreakdown + stateTransitions), confidence (populated as zeros, scored later by ConfidenceEngine).
